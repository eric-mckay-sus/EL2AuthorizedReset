AdminInterface
=========

Purpose
---------

`AdminInterface` is a Blazor-based application that provides a UI to help admins add and remove associates from the system, as well as add or remove lines per associate.

How it works
---------

1. 

Run locally
---------

From the repo root run:

```bash
dotnet run --project AdminInterface\AdminInterface.csproj
```

Configuration
---------

- `appsettings.json` and `appsettings.Development.json` in the project root contain the information required to connect to the database.
- Use `Properties/launchSettings.json` for development launch configurations.
